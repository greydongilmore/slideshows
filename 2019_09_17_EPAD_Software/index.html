EPAD Analysis Software

Developers: Greydon Gilmore & Wafiq Seyed

Software Overview

  • Data import

  • Settings Panel

  • Plotting

Data Import

  • The selected directory contains subdirectories for all subjects

  • Stimulation/Recording info read from SSEP-Settings.txt

Settings Panel

  • Sampling Freq: freely change sample frequency

  • Filters: butterworth band-pass/notch filter

  • Threshold: max threshold for noise

  • Num Avgs: number of waveforms to average

  • Plot: real-time update of plot, plot saved into subject directory folder
